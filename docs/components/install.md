# 安装

## NPM包 安装

```bash
  npm install holo-ui

  # or

  yarn add holo-ui
```

## CDN 安装

```html
<!-- 引入样式 -->
<link rel="stylesheet" href="https://localhost:8080/holo-ui/lib/index/index.css">
<!-- 引入组件库 -->
<script src="https://localhost:8080/holo-ui/lib/index/index.js"></script>
```

如果是通过 npm包 安装，并希望配合 webpack 使用，请阅读下一节：快速上手。
