# loading 加载

### 基础用法


### Attributes
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| text    |  显示在加载图标下方的加载文案  |   string   |   -   |     -    |
| type    |  类型  |   string   |   global / local   |     global    |
| theme   |  主题色    | string   |  blue / white  | blue   |
| lock    |  是否加锁    | boolean   |  true / false  | true   |
| maskType    |  遮罩类型,只有lock:true时生效    | string   |  transparent / white / black  | white   |
